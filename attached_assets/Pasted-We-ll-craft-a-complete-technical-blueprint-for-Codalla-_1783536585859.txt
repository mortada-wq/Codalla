🧠 COdalla – The Ultimate AI Coding Wrapper
Prompt for Replit AI Agent: Build This Step-by-Step
Goal: Create a full-stack web application on Replit (Node.js + React) that acts as an online IDE with integrated AI coding assistants, GitHub repo sync, multi-model AI chat, code fix/diff view, and enterprise-grade features, using only your own API keys (SiliconFlow, OpenRouter, etc.). No recurring tool fees – you pay only for token usage.

1. OVERVIEW & ARCHITECTURE
Tech Stack (Replit-optimized):

Frontend: React (Vite), TailwindCSS, Monaco Editor (code editor), React Router, React Query

Backend: Express.js (Node.js) or Next.js API routes (if using Next.js for SSR)

Database: SQLite (via better-sqlite3) or Replit Database (key-value) – store user data, projects, API keys, conversation history

Real-time: Socket.IO for live collaboration / streaming AI responses

Git Integration: isomorphic-git + GitHub REST API (for pull, push, commit, branch)

AI Providers:

SiliconFlow API (DeepSeek V3/V4, GPT-120, etc.)

OpenRouter API (any model, including Gemma 26B/32B)

Unified OpenAI-compatible endpoint interface

Authentication (optional enterprise): Passport.js (GitHub OAuth) or simple API key login

Containerization/Sandbox: Replit’s native environment – for running code, use Replit’s Nix modules or Docker (if possible) or just a backend executor with child_process (secured)

Deployment: Within Replit, always-on repl, with secret environment variables for API keys.

3. DATABASE SCHEMA (SQLite)
sql
-- Users (simple auth)
CREATE TABLE users (
  id TEXT PRIMARY KEY,
  username TEXT UNIQUE,
  email TEXT,
  github_token TEXT ENCRYPTED,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- API Keys (encrypted)
CREATE TABLE api_keys (
  id TEXT PRIMARY KEY,
  user_id TEXT REFERENCES users(id),
  provider TEXT, -- 'siliconflow', 'openrouter', etc.
  key_value TEXT ENCRYPTED,
  label TEXT,
  is_active BOOLEAN DEFAULT 1
);

-- Projects
CREATE TABLE projects (
  id TEXT PRIMARY KEY,
  user_id TEXT REFERENCES users(id),
  name TEXT,
  local_path TEXT, -- path inside repl filesystem
  git_remote_url TEXT,
  current_branch TEXT DEFAULT 'main',
  last_synced DATETIME
);

-- AI Conversations
CREATE TABLE conversations (
  id TEXT PRIMARY KEY,
  project_id TEXT REFERENCES projects(id),
  model_name TEXT,
  created_at DATETIME
);

CREATE TABLE messages (
  id TEXT PRIMARY KEY,
  conversation_id TEXT REFERENCES conversations(id),
  role TEXT, -- 'user', 'assistant', 'system'
  content TEXT,
  tokens_used INTEGER,
  cost REAL,
  timestamp DATETIME
);

-- Usage log
CREATE TABLE usage_log (
  id TEXT PRIMARY KEY,
  user_id TEXT,
  model TEXT,
  prompt_tokens INTEGER,
  completion_tokens INTEGER,
  cost REAL,
  timestamp DATETIME
);
